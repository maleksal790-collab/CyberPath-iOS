# Release Notes

## 1.0 Build 1

Initial native iOS package for CyberPath.

Included:

- SwiftUI dashboard with explained recommendations.
- Learning domains and topic detail screens.
- Topic notes, bookmarks, completion, and quizzes.
- Study session logging.
- Scenario lab with saved decision evidence.
- Interactive visual studio.
- Cross-domain diagnostic.
- Evidence report with non-certification disclaimer.
- Local progress export/import.
- Governed delivery pipeline and release documentation.

Known release boundary:

- Final TestFlight/App Store deployment requires macOS, Xcode, signing credentials, and App Store Connect access.
