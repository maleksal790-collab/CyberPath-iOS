# Privacy Nutrition Draft

## Data Collection Position

CyberPath iOS does not include analytics, advertising SDKs, account creation, cloud synchronization, or backend network services.

## User Data

The app stores learning progress, notes, quiz scores, diagnostic results, and scenario evidence locally on the device.

## Exported Data

Progress export is initiated by the user. Exported JSON may contain learner notes and scenario rationale. Treat exported files as private user-controlled files.

## App Store Privacy Draft

- Data linked to user: none, unless future account features are added.
- Data used to track user: none.
- Data collected by developer: none in the current local-first implementation.

Reconfirm these answers before App Store submission if any network, analytics, crash-reporting, AI, or account feature is added.
