# CyberPath iOS: Master Documentation

## 1. Project Overview
**CyberPath iOS** is a native SwiftUI iOS version of the CyberPath learning app. 

## 2. Security & Privacy Posture
- No Account Required: Learners access all content anonymously.
- No Analytics: No third-party tracking.
- Local Storage: All progress is stored locally via UserDefaults.

## 3. Delivery Pipeline & Governance
Changes follow a governed delivery process using the included `Makefile` and GitHub Actions.

## 4. Test Strategy & Quality Gates
Manual acceptance required for functional, scenario, and accessibility gates.

## 5. Deployment & Release Process
Deployment requires a macOS machine with Xcode 15+ and an authorized Apple Developer account.
